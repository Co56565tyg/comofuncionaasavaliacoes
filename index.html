Como funcionam as avaliações das empresas?

O Reclame AQUI é como um juiz que avalia as empresas com base no que as pessoas dizem sobre elas. Ele olha para algumas coisas importantes:

Índice de Resposta (IR): Isso mostra quantas reclamações as empresas responderam. Se elas não responderem, é como se estivessem ignorando as pessoas.

Média das Avaliações (MA): As pessoas dão notas de 0 a 10 para dizer o quão satisfeitas estão com o atendimento. O Reclame AQUI calcula a média dessas notas.

Índice de Solução (IS): Mostra quantas vezes as empresas resolveram os problemas das pessoas. Se elas não resolverem, é um sinal de que não estão ajudando.

Índice de Novos Negócios (IN): Isso mostra se as pessoas ainda querem fazer negócios com a empresa depois de terem uma reclamação. Se elas não quiserem, é um sinal de que algo está errado.

O Reclame AQUI usa essas informações para calcular uma nota para cada empresa. Ele usa uma fórmula que dá mais importância para algumas coisas do que para outras:

Responder reclamações é muito importante, então tem peso 2.

A média das notas dos clientes tem peso 3, porque a satisfação deles é essencial.

Resolver os problemas das pessoas também é muito importante, então tem peso 3.

E saber se as pessoas ainda querem fazer negócios com a empresa tem peso 2.

Depois de calcular tudo isso, o Reclame AQUI dá uma nota final para a empresa. Se a nota for alta, é ótimo! Se for média, é bom. Se for baixa, é ruim. E se for muito baixa, é melhor evitar essa empresa.

Mas o Reclame AQUI só usa essa fórmula se a empresa tiver respondido pelo menos metade das reclamações e se tiver recebido pelo menos 10 avaliações.

Se não, ela é automaticamente considerada como "Não Recomendada". Isso é para garantir que só empresas que tratam bem os clientes recebam uma boa nota.
